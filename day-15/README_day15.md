## 🚀 Sample Snippet – Using .push() and .pop()

```js
const fruits = ["apple", "banana"];
fruits.push("orange");  // Adds 'orange' at the end
fruits.pop();           // Removes the last element
console.log(fruits);    // Output: ['apple', 'banana']
```
