## 🚀 Sample Snippet – Conditionals

```js
let score = 85;
if (score >= 90) {
  console.log("Grade: A+");
} else if (score >= 80) {
  console.log("Grade: A");
} else if (score >= 70) {
  console.log("Grade: B");
} else {
  console.log("Grade: C or below");
}
```

## 🛠 How to Run

You can run JavaScript files using:

